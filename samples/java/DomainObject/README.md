### Introduction

This sample library is a common library used by all samples for domain objects used in Northwind. Sample contains following domain objects:

1. Category
2. Customer
3. Order
4. OrderDetails
5. Products

> **Note:** If you are unfamiliar with Git and GitHub, you can download the entire collection as a 
> [ZIP file](../../archive/master.zip).

### Prerequisites

Before the sample application is used make sure that:

- Build the sample.
